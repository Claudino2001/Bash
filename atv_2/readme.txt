Construa um primeiro script que receba como argumento o nome de um arquivo e verifique se este arquivo é um arquivo padrão, um diretório, ou executável. Encerre o script com os seguintes códigos de retorno:

rc=0 para Arquivo padrão;
rc=1 para Diretório;
rc=2 para Executável.

Construa um segundo script que receba como argumento o nome de um arquivo, chame o primeiro script para verificação do tipo deste arquivo e, com base no seu código de retorno, exiba o tipo do arquivo.
